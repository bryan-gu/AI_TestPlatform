用户登录注册需求说明书

文档编号：REQ-AUTH-001  |  版本：v1.2  |  作者：李明  |  最后更新：2026-03-28

1. 概述

本文档定义了电商平台用户登录注册模块的功能需求，包括手机号注册、邮箱注册、第三方登录（微信/支付宝）以及密码找回等功能。

2. 功能需求

2.1 手机号注册

用户输入手机号 → 发送短信验证码（60s 有效期）→ 输入验证码 → 设置密码（8-20位，需含大小写字母+数字）→ 注册成功。

业务规则：同一手机号不可重复注册；验证码错误超过 5 次锁定 30 分钟；密码不可与手机号后 6 位相同。

2.2 第三方登录

支持微信 OAuth 2.0 和支付宝授权登录。首次第三方登录需绑定手机号。

  • 微信登录：调用微信开放平台 API，获取 openid 和 unionid
  • 支付宝登录：调用支付宝 auth 接口，获取 user_id
  • 绑定流程：第三方授权 → 跳转绑定手机号页面 → 短信验证 → 绑定完成

3. 非功能需求

指标 要求
接口响应时间≤ 500ms (P95)
并发支持≥ 1000 QPS
短信到达率≥ 99.5%
密码加密bcrypt, salt rounds ≥ 12

4. 接口定义

{

}

}
